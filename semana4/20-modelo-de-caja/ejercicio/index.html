Ejercicio #1

  1. Crear una caja padre con las siguientes características:
    1. 500px de alto
    2. 500px de ancho
    3. Borde discontinuo (dashed)
  2. Crear 4 cajas hijos con las siguientes características:
    1. Cada una de las 4 cajas tendrá un padding de 10px
  3. Dar características a las cajas hijo de la siguiente manera:
    1. Caja 1 debe de tener una imagen de fondo(background-image)
    2. Caja 2 debe de tener un fondo 'red'
    3. Caja 3 debe de tener un color de fondo 'salmon'
    4. Caja 4 debe de tener un color de fondo 'green'
    5. Caja 1 y caja 2:
      1. Borde superior e inferior de 10px de color 'grey'
    6. Caja 3 y caja 4:
      1. Borde derecho e izquierdo de 10px color 'cyan'
  4. El ejercicio debe de cumplir con los siguientes requisitos
    1. No se puede utilizar la propiedad overflow
    2. No se puede utilizar la propiedad overflow
    3. No se puede utilizar el modelo de caja alternativo
    4. Las 4 cajas hijo deben de tener las mismas medidas de alto y de ancho(sumando bordes y paddings) ocupando cada caja 1/4 del espacio del contenedor padre
    5. Todas las medidas tienen que ser asignadas en pixeles
