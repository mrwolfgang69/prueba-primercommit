Tipografía

Las tipografías (también llamadas fuentes) son una parte muy importante dentro del mundo de CSS, ya que son uno de los pilares del diseño web. Por ello, es de suma importancia elegir correctamente la tipografía, su tamaño, color, espacioado entre líneas y letras, la persona, ya que esto determinará si el usuario accede o no a los contenidos de una página.

Enlaces

A la hora de estilar los enlaces, es importante comprender cómo utilizar las pseudoclases para diseñar los estados de un enlace:

  • Link (no visitado): El estado predeterminado, se puede modificar usando :link
  • Visited: Hace referencia a un enlace ya visitado y se puede estilar utilizando :visited
  • Hover: Se utiliza para estilar un elemento cuando el mouse es pasado por encima, se utiliza :hover
  • Focus: Cuando damos tab y se selecciona un enlace, podemos estilarlo usando :focus
  • Active: A la acción de dar clic sostenido sobre un enlace también se puede estilar, utilizando :active
