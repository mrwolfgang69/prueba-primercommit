¿Qué es CSS?

Las siglas CSS (Cascading Style Sheets) significan "Hojas de estilo en Cascada" y parten de un concepto simple pero muy potente: aplicar estilos (colores, formas, márgenes, etc...) a uno o varios documentos HTML de forma masiva.

Conceptos Básicos

¿Qué es la sintaxis CSS?

Se considera sintaxis al orden y relación de los elementos que componen una correcta instrucción.

¿Qué es una regla CSS?

Una regla CSS es la forma adecuada de escribir una instrucción mediante código CSS, es decir, la manera en que se debe escribir código CSS para que éste sea interpretado y aplicado correctamente por el navegador web.

Anatomía de una regla CSS

Una regla CSS está formada principalmente por uno o más selectores y una o más declaraciones, la o las declaraciones siempre van dentro de llaves, cada declaración se compone de la propiedad con su respectivo valor.

Tipos de selectores

Selector universal

Este selector en CSS selecciona todos los elementos de un documento

Selector de tipo

Los selectores de tipo en CSS se refieren a los selectores que apuntan a un tipo de elemento HTML

Selectores de id, atributo y clase

Este tipo de selectores en CSS apuntan a un elemento HTML identificado, es decir, a un elemento HTML que tenga el atributo id o class.

Tanto el selector de tipo id, atributo y class funcionan de la misma manera; la única diferencia es que para los selectores de tipo id se le antepone el signo de numeral (#), para los selectores de atributo corchetes ([]) y a los de tipo class un punto (.)

Operadores de combinación de selectores

Con ayuda de los operadores de combinación puedes combinar otros selectores con el fin de delimitar elementos de nuestros documentos.

Hijos directos

El selector «mayor qué» (símbolo «>») es utilizado en CSS para seleccionar todos los elementos que sean directamente descendientes de otro, es decir, que sean hijos directos de un determinado elemento padre, por eso también se conoce como «selector de hijos«.

Hijos descendientes

Para seleccionar a todos los descendientes del contenedor padre no es necesario el selector «mayor qué», sólo es necesario colocar el selector padre seguido del elemento descendiente que quieres estilar.
