Colores

Los colores en CSS se pueden indicar de cinco formas diferentes: palabras clave, colores del sistema, RGB hexadecimal, RGB numérico y RGB porcentual. Aunque el método más habitual es el del RGB hexadecimal.

Palabras Clave

Son palabras ya incluidas en CSS para seleccionar colores, sólo con colocar la palabra es suficiente para estilar con color un elemento.

  • red
  • lime
  • blue
  • fuchsia
  • gray
  • teal

RGB hexadecimal

Para obtener el RGB de los colores en hexadecimal se eligen números o letras que van desde el 0 - f, es decir, 0123456789abcdef, donde 0 es el más oscuro y f el más claro.

  • red
  • lime
  • blue
  • fuchsia
  • gray
  • teal

RGB numérico

Para obtener el RGB de los colores en númerico se eligen números que van desde el 0 - 255, donde 0 es el más oscuro y 255 el más claro.

  • red
  • lime
  • blue
  • fuchsia
  • gray
  • teal

RGB porcentual

Para obtener el RGB de los colores en porcentaje van desde el 0% - 100%, donde 0% es el más oscuro y 100% el más claro.

  • red
  • lime
  • blue
  • fuchsia
  • gray
  • teal

Colores RGBa

Los colores en RGB tienen opacidad, para ello está RGBa, se puede utilizar el valor número o porcentual para definir la opacidad, para integrar la opacidad debe ir al último como un valor extra del RGB.

  • red
  • lime
  • blue
  • fuchsia
  • gray
  • teal

Colores HLS

Es un modo de expresar los colores mediante 3 componentes: el «tono (Hue)», la «saturación (Saturation)» y el «luminosidad （Lightness / Luminance)».

  • red
  • lime
  • blue
  • fuchsia
  • gray
  • teal

Colores HLSa

Al igual que los colores en RGB, los colores en HLS tienen opacidad, para ello está HLSa, se puede utilizar el valor de 0 - 1 para definir la opacidad, para integrar la opacidad debe ir al último como un valor extra del HLS.

  • red
  • lime
  • blue
  • fuchsia
  • gray
  • teal
